# Taskmesh Design Document

## Introduction
### What is Taskmesh?
Taskmesh is a distributed task manager for teams. 


## User stories
### Tasks
- User creates a tasklists. The tasklist has a name.
- User creates a tasks in a tasklist. The task has a: 
    - name,
    - description,
    - creation date,
    - completion status and, 
    - completion status.

### Teams
- User creates a team. That user is a team leader.
- User invites another user to a team. That user could be a
leader or a member.
